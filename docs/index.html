IoT Project

智慧影像監控系統

以 Raspberry Pi 與相機模組為核心，建構可即時辨識人物或目標物件的智慧監控原型， 並在偵測到目標後進行警示與影像紀錄。

目標

建立一套可即時監控環境畫面的智慧影像監控系統，利用影像辨識技術判斷畫面中是否出現人物或特定物件， 並在偵測到目標時立即執行警示與紀錄功能，作為智慧居家與安全監控的基礎原型。

材料

  • Raspberry Pi 4 / 5
  • Raspberry Pi Camera 模組或 USB Webcam
  • LED 指示燈
  • 蜂鳴器
  • 伺服馬達（進階擴充，可選）
  • Python 與影像辨識模型環境

系統功能

  • 即時影像擷取
  • 人物或目標物件辨識
  • 偵測到目標時啟動燈光或蜂鳴器警示
  • 擷取事件畫面並保存
  • 可延伸為雲端通知或鏡頭追蹤系統

流程架構

相機持續擷取監控畫面
↓
Raspberry Pi 接收影像資料
↓
影像辨識模型分析畫面內容
↓
判斷是否出現人物或指定物件
↓
若偵測成功，啟動警示並保存影像
↓
將結果顯示於監控介面
