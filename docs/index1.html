人物標記系統

多節點影像共享與標記 HUD 系統

目標 系統功能 流程架構 材料

專題目標

  • 建立多個攝影裝置（2 節點）同步影像系統
  • 即時辨識每個畫面中的人位置
  • 假如被掩蔽 標示由紅變黃再變綠
  • 整合所有節點資訊並共享至系統
  • 在個人 HUD(用電腦螢幕模擬) 上顯示人物標記

系統功能

  • 多攝影節點： 支援 N 個 ESP32-CAM 同時上傳影像
  • 敵人偵測： 每個節點畫面進行辨識
  • 位置標記： 在畫面中標示人 bounding box
  • 資訊整合： 將所有節點結果集中處理
  • HUD 顯示： 在個人螢幕顯示人位置（模擬戰術介面）

流程架構

材料與設備

  • ESP32-CAM（2個， 或替代品）
  • Raspberry Pi 4 / 5
