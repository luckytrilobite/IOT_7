攝影機串流與事件通知平台

即時影像串流與智慧事件感知系統

目標 系統功能 流程架構 材料

專題目標

  • 建立即時攝影機串流系統
  • 分析影像並產生事件（movement / detection）
  • 將事件即時推送至後端系統
  • 建立可視化儀表板顯示影像與事件
  • 實現事件通知（警告 / 提示）

系統功能

  • 即時影像串流： 支援 Webcam 或 Raspberry Pi Camera
  • 動態偵測： 偵測畫面中是否有移動物體
  • 事件產生： 將偵測結果轉換為事件（JSON）
  • 事件通知： 即時推送至前端或訊息系統
  • 資料記錄： 將事件與截圖儲存於資料庫
  • Dashboard： 顯示影像串流與事件列表

流程架構

Camera
↓
影像串流（HTTP / RTSP）
↓
AI / OpenCV 分析
↓
事件產生（MQTT / HTTP）
↓
Server（Flask / Django）
↓
Dashboard / 通知系統

材料與設備

  • Raspberry Pi 4 / 5
  • USB Webcam 或 Pi Camera
  • Python（OpenCV / Flask）
  • MQTT Broker（Mosquitto）
  • 資料庫（MariaDB）
